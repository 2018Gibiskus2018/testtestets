Создавайте интерактивные уроки.

Есть маленькая хитрость: Mischief. Представляет надстройку Mouse Mischief для PowerPoint с одновременной поддержкой нескольких мышей.

Mouse Mischief интегрируется в Microsoft PowerPoint (2007-2013) и позволяет добавлять в слайды опросы, тестовые задания и поля для рисования. Во время урока учащиеся выбирают правильные ответы, рисуют каждый при помощи собственной мышки.

Mouse Mischief дает новые возможности работы в классе, позволяя учителю создавать интерактивные презентации увлекающие каждого ученика с поддержкой нескольких мышей. До 25 учеников, каждый с помощью собственной мышки, отвечают на вопросы и рисуют на общем экране. И что очень важно, приложение распространяется бесплатно.
